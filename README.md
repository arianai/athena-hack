# Refill
A video chat web application using the Vonage Video API.

## Stack
HTML,
CSS,
JavaScript,
Node,
Express

## Installation
1. Open project in your editor
2. In termninal run: 
```
npm install
```
3. Then run:
```
npm run dev
```
4. Open your browser and enter localhost:300 in the URL bar
